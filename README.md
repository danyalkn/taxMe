# Chrome Extension 
Project Name: taxMe <br />
Project Date: April 2020 <br /> 
Technologies Used: HTML, CSS, Javascript<br />
Project Description: Due to Covid-19, online shopping has become the go to way to shop. This quick and easy chrome extension alls you to enter a price and will return it with the resepective price with tax included. This is much more efficient and less tediousness method then adding the product to your cart, then going to checkout and then being able to see the tax included price.
